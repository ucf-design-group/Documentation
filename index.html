Welcome!

The purpose of this site is to provide documentation for some of the wild things you might encounter while working as a Web Designer at OSI. We work in many environments - from Wordpress in PHP to CreativeQ in Ruby to server administration via the Terminal - and we don't expect you to know everything.

Most of the original documentation was written by AJ Foster (me), one of the old web developers. If you have trouble with anything you read here, please feel free to e-mail me. You can find a link to my e-mail above and below.
